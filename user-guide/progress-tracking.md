Monitor your learning journey across all courses and modules.

## Course Progress

### How It's Calculated

```
Course Progress = (Completed Modules / Total Modules) × 100%
```

**Example:**
- Course has 10 modules
- You've completed 4 modules
- Progress = 40%

### Progress States

| Status | Meaning |
|--------|---------|
| Not Started | 0% - No modules accessed |
| In Progress | 1-99% - Some modules complete |
| Completed | 100% - All modules complete |

## Module Progress

### Completion Criteria

Different module types have different completion requirements:

| Module Type | Completion Requirement |
|-------------|----------------------|
| Video | Watch to end (or mark complete) |
| Text | Scroll through (or mark complete) |
| Quiz | Pass with required score |
| Download | Access/download resource |
| Assignment | Submit and receive grade |

### Video Progress

For video lessons:
- **Position saved**: Resume where you left off
- **Completion threshold**: 90% watched = complete
- **Watch percentage**: Tracked as you watch

### Quiz Progress

For quiz modules:
- **Attempts tracked**: Number of tries
- **Best score**: Your highest score
- **Last score**: Most recent attempt
- **Pass status**: Passed/Not Passed

## Sequential Progress

### Module Locking System

Modules unlock in order:
1. First module always unlocked
2. Complete module to unlock next
3. Quiz pass required for progression
4. Can't skip ahead

### Benefits:
- Ensures foundational knowledge
- Prevents overwhelming content
- Optimizes learning retention

## Resetting Progress

### Can You Reset?
Currently, progress cannot be reset by users. This is by design to:
- Maintain accurate learning records
- Support certification tracking
- Ensure completion integrity

### Need to Restart?
Contact support if you have a legitimate need to reset:
- Account issues
- Course updates requiring fresh start
- Special circumstances

## Troubleshooting

### Progress Not Updating

1. **Wait a moment**: Sync can take a few seconds
2. **Refresh the page**: Force data reload
3. **Check connection**: Ensure stable internet
4. **Clear cache**: Browser cache may be stale
5. **Contact support**: If issue persists

### Incorrect Progress Shown

1. Log out and back in
2. Try a different browser
3. Check if module is truly complete
4. Report specific discrepancy to support

### Lost Progress

Progress is stored server-side and shouldn't be lost. If you experience this:
1. Document what's missing
2. Note the course and modules affected
3. Contact support immediately
4. We can often restore from backups
