## Bitcoin Retirement Calculator

### Overview
A powerful tool to project your crypto investment growth.

### Features

**Two Calculation Modes:**

| Mode | Description |
|------|-------------|
| **Conservative** | Lower projected returns, safer estimates |
| **Optimized** | Higher potential returns with more risk |

### How to Use

1. Enter your current age
2. Enter your target retirement age
3. Input current BTC holdings (or investment amount)
4. Select calculation mode
5. View projected growth

### What You'll See
- **Chart**: Visual growth projection
- **Table**: Year-by-year breakdown
- **Summary**: Key milestones
- **BTC Price**: Real-time pricing

### Assumptions
- Historical BTC growth patterns
- Compound growth over time
- No additional investments (or specify additions)
- Current market conditions

> **Note**: Projections are for educational purposes. Actual results will vary.
