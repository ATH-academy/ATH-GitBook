# Quizzes & Assessments

Test your knowledge and validate your learning with our quiz system.

---

## Overview

Quizzes are integrated throughout courses to:
- Reinforce learning
- Test comprehension
- Unlock subsequent modules
- Contribute to certification

---

## Quiz Types

### Short Quizzes
- **3 questions** per quiz
- **Pass requirement**: 2/3 correct (67%)
- **Purpose**: Quick knowledge check after lessons

### Standard Quizzes
- **10 questions** per quiz
- **Pass requirement**: 8/10 correct (80%)
- **Purpose**: Comprehensive module assessment

### Certification Exams
- **25+ questions**
- **Pass requirement**: Varies by certification
- **Purpose**: Final credential validation

---

## Taking a Quiz

### Starting a Quiz
1. Navigate to the quiz module
2. Read any instructions provided
3. Click **"Start Quiz"**
4. Timer begins (if time-limited)

### During the Quiz
- Read each question carefully
- Select your answer(s)
- Click **"Next"** to proceed
- Click **"Previous"** to review (if allowed)
- Click **"Submit"** when finished

### Question Format
All questions are multiple choice:
- Single correct answer
- Read all options before selecting
- Look for key terms from lessons

---

## Quiz Features

### Answer Shuffling
- Questions may appear in random order
- Answer options may be shuffled
- Prevents memorization patterns

### Time Limits
Some quizzes have time limits:
- Timer displayed at top
- Warning when time is low
- Auto-submit when time expires

### Retry Capability
- Most quizzes allow unlimited retries
- Some may have a retry limit
- Wait period may apply between attempts

---

## Scoring

### How Scores are Calculated

```
Score = (Correct Answers / Total Questions) × 100%
```

### Score Tracking

| Metric | Description |
|--------|-------------|
| **Best Score** | Your highest score ever |
| **Last Score** | Most recent attempt |
| **Attempts** | Number of times taken |
| **Passed** | Yes/No based on threshold |

### Pass Thresholds

| Quiz Type | Pass Requirement |
|-----------|-----------------|
| 3-Question | 2/3 (67%) |
| 10-Question | 8/10 (80%) |
| Certification | 80-85% (varies) |

## Quiz Results

### Immediate Feedback
After submitting:
1. See your score instantly
2. View pass/fail status
3. Review correct answers (if enabled)
4. See explanations for answers

### Results Display
```
Quiz Complete!

Score: 8/10 (80%)
Status: ✓ Passed

Correct: Questions 1, 2, 3, 5, 6, 7, 9, 10
Incorrect: Questions 4, 8

[Review Answers] [Continue to Next Module]
```

### What Happens Next

**If Passed:**
- Module marked as complete
- Next module unlocks
- Progress updates
- Continue learning

**If Failed:**
- Review material
- Retry when ready
- Module remains incomplete
- Next module stays locked

## Best Practices

### Before the Quiz
1. **Complete all lessons**: Watch/read all content
2. **Take notes**: Review your notes before
3. **Understand concepts**: Don't just memorize
4. **Check examples**: Review case studies

### During the Quiz
1. **Read carefully**: Don't rush
2. **Eliminate wrong answers**: Narrow options
3. **Use logic**: Apply what you learned
4. **Trust yourself**: First instinct often correct
5. **Manage time**: Don't get stuck on one question

### If You Fail
1. **Don't stress**: Retakes are allowed
2. **Review material**: Focus on weak areas
3. **Identify gaps**: What did you miss?
4. **Wait if needed**: Let information settle
5. **Try again**: Persistence pays off

## Quiz Settings

### Configured by Admins

Each quiz may have different settings:

| Setting | Description |
|---------|-------------|
| Passing Score | Minimum % to pass |
| Max Attempts | How many retries allowed |
| Time Limit | Maximum time to complete |
| Shuffle Questions | Random question order |
| Shuffle Answers | Random answer order |
| Show Results | Whether to show correct answers |

### Viewing Quiz Info
Before starting, you can see:
- Number of questions
- Time limit (if any)
- Required score to pass
- Number of attempts remaining

## Troubleshooting

### Quiz Won't Load
1. Refresh the page
2. Clear browser cache
3. Try different browser
4. Check internet connection
5. Contact support

### Answers Not Saving
1. Don't close browser mid-quiz
2. Check for stable connection
3. Use "Save Progress" if available
4. Complete quiz in one session

### Score Discrepancy
If you believe your score is wrong:
1. Note the quiz and questions
2. Take screenshots if possible
3. Contact support with details
4. We'll review and correct if needed

### Timer Issues
If timer seems wrong:
1. Don't refresh during quiz
2. Check system clock accuracy
3. Report any significant issues

## FAQ

**Q: Can I pause a quiz?**
A: Not for timed quizzes. Untimed quizzes may allow leaving and returning.

**Q: What if I accidentally close the browser?**
A: Progress may be lost. Complete quizzes in one session.

**Q: Can I see the correct answers?**
A: Depends on quiz settings. Some show answers after passing.

**Q: Do failed attempts count against me?**
A: No, only your best score is used for certification.

**Q: How long should I wait before retrying?**
A: Review the material first. No minimum wait required unless specified.
