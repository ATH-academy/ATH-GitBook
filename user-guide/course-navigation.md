# Course Navigation

Learn how to navigate and access courses on All Time High Academy.

---

## Accessing Courses

### From the Dashboard
1. Log into your account
2. Click **"Courses"** in the main navigation
3. Browse the course catalog
4. Click any course to view details

### From Your Dashboard
- **Recent Courses**: Quick access to courses you've started
- **Continue Learning**: Resume your last active course
- **Progress Cards**: See completion status at a glance

## Course Structure

Courses are organized hierarchically:

```
Course
├── Module 1
│   ├── Lesson 1.1
│   ├── Lesson 1.2
│   └── Quiz 1
├── Module 2
│   ├── Lesson 2.1
│   ├── Lesson 2.2
│   └── Quiz 2
└── Final Assessment
```

### Components

| Component | Description |
|-----------|-------------|
| **Course** | The complete learning package |
| **Module** | A chapter or section of the course |
| **Lesson** | Individual learning units (video/text) |
| **Quiz** | Knowledge assessment after modules |

## Sequential Module Unlocking

### How It Works
Modules unlock in sequence to ensure proper learning progression:

1. **First module** is always unlocked
2. Complete the current module to unlock the next
3. Quizzes must be passed to mark module complete
4. Progress is tracked automatically

### Visual Indicators

| Icon | Meaning |
|------|---------|
| Open lock | Module unlocked and accessible |
| Closed lock | Module locked (complete previous first) |
| Checkmark | Module completed |
| Progress bar | Partially completed |

### Locked Module Message
When clicking a locked module:
> "Complete the previous module to unlock this content."

## Course Progress

### How Progress is Calculated

```
Course Progress = (Completed Modules / Total Modules) × 100%
```

### Marking Modules Complete

Modules are marked complete when:
1. **Video lessons**: Watched to completion (or marked manually)
2. **Text content**: Scrolled through (or marked manually)
3. **Quizzes**: Passed with required score
4. **Downloads**: Accessed at least once

### Manual Completion
For non-quiz modules, you can manually mark as complete:
1. Click the module
2. Look for "Mark as Complete" button
3. Click to toggle completion status

## Tips for Success

### Optimal Learning
1. **Dedicated time**: Set aside focused study periods
2. **Notes**: Take notes in your dedicated notebook
3. **Practice**: Apply concepts immediately
4. **Review**: Revisit challenging content
5. **Quizzes**: Don't skip assessments

### Technical Tips
- Use a stable internet connection for video
- Chrome, Firefox, or Brave browsers recommended
- Clear cache if videos don't load
- Try hard refresh (Ctrl+Shift+R) for issues

---

## Troubleshooting

### Video Won't Play
1. Check internet connection
2. Try a different browser
3. Clear browser cache
4. Disable VPN if active
5. Contact support if persists

### Progress Not Saving
1. Don't close browser during video
2. Check for stable connection
3. Refresh the page
4. Log out and back in
5. Report to support with details

### Module Won't Unlock
1. Verify previous module is 100% complete
2. Check if quiz was passed
3. Refresh the page
4. Contact support with course name
