# Troubleshooting

Solutions to common technical issues.

---

## Login Issues

### Can't log in with email/password

**Symptoms:**
- "Invalid credentials" error
- Login button doesn't respond

**Solutions:**
1. Double-check email spelling
2. Use "Forgot Password" to reset
3. Check for caps lock
4. Clear browser cache and try again
5. Try incognito/private window

### Discord login not working

**Symptoms:**
- Popup closes without logging in
- Error after authorizing

**Solutions:**
1. Make sure you're logged into Discord
2. Try logging out of Discord and back in
3. Clear browser cookies for Discord
4. Try a different browser
5. Disable browser extensions

### Whop login not working

**Symptoms:**
- Redirect fails
- "Authorization error"

**Solutions:**
1. Ensure you have a Whop account
2. Check you're logged into Whop
3. Clear cookies and try again
4. Contact support if persistent

## Video Playback Issues

### Video won't load

**Symptoms:**
- Blank video player
- Spinning loading indicator
- Error message

**Solutions:**
1. Refresh the page
2. Check internet connection (try other sites)
3. Clear browser cache
4. Try a different browser
5. Disable ad blockers
6. Disable VPN

### Video plays but buffers constantly

**Symptoms:**
- Frequent pauses
- Low quality despite good internet

**Solutions:**
1. Close other tabs/applications using bandwidth
2. Connect to faster network
3. Lower video quality if available
4. Try different time of day

### Video quality is poor

**Symptoms:**
- Blurry video
- Pixelation

**Solutions:**
- Video automatically adjusts to your connection
- Ensure stable, fast internet
- Close other bandwidth-heavy applications

### No audio

**Symptoms:**
- Video plays but silent

**Solutions:**
1. Check video isn't muted
2. Check system volume
3. Check browser isn't muted
4. Try different headphones/speakers
5. Refresh the page

## Progress Issues

### Progress not saving

**Symptoms:**
- Completed module shows as incomplete
- Progress resets after refresh

**Solutions:**
1. Stay on page until video completes
2. Don't close browser mid-lesson
3. Check internet connection
4. Refresh and wait a moment
5. Log out and back in
6. Report specific module to support

### Module stuck on incomplete

**Symptoms:**
- Watched entire video but not marked complete
- Can't proceed to next module

**Solutions:**
1. Manually mark as complete (if available)
2. Watch last 30 seconds again
3. Refresh and try again
4. Clear cache and retry
5. Contact support with module name

### Quiz score not recorded

**Symptoms:**
- Passed quiz but module shows incomplete
- Score shows as 0

**Solutions:**
1. Don't close browser during quiz
2. Ensure stable internet
3. Retake the quiz
4. Contact support with quiz details

## Module/Course Access

### Module is locked

**Symptoms:**
- Can't access module
- Lock icon displayed

**This is expected behavior if:**
- Previous module isn't complete
- Previous quiz wasn't passed
- Content requires higher access level

**If incorrectly locked:**
1. Verify previous module is truly complete
2. Check quiz scores
3. Refresh the page
4. Contact support

### Course not accessible

**Symptoms:**
- "Upgrade required" message
- Content not loading

**Check:**
1. Your subscription status
2. Required access level for course
3. If subscription payment is current
4. Log out and back in

## Account Issues

### Can't verify email

**Symptoms:**
- No verification email received
- Verification link expired

**Solutions:**
1. Check spam/junk folder
2. Click "Resend Verification"
3. Wait 5 minutes and check again
4. Add our domain to contacts
5. Contact support for manual verification

### Profile changes not saving

**Symptoms:**
- Changes revert after refresh

**Solutions:**
1. Click "Save" button
2. Wait for confirmation message
3. Check internet connection
4. Try different browser
5. Contact support

## Discord Connection

**Symptoms:**
- Connection fails

**Solutions:**
1. Make sure logged into Discord
2. Check Discord account is verified
3. Try different browser
4. Revoke and reconnect
5. Contact support

### Browser Settings

For best experience:
- Enable JavaScript
- Allow cookies for alltimehigh.academy
- Disable aggressive ad blockers
- Keep browser updated

### Clearing Cache

**Chrome:**
1. Settings > Privacy > Clear browsing data
2. Select "Cached images and files"
3. Click Clear Data

**Firefox:**
1. Settings > Privacy > Cookies and Site Data
2. Click Clear Data

**Safari:**
1. Preferences > Privacy
2. Click Manage Website Data > Remove All

1. **Document the problem:**

   - What happened instead?
   - Any error messages?
   - Screenshots if possible

2. **Contact support:**
   - Email: support@alltimehigh.academy
   - Discord: #support channel
   - Include all details from step 1

3. **Response time:**
   - Free/Community: 24-48 hours
   - VIP: Same day
   - Concierge: Priority response
